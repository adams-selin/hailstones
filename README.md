# hailstones
Notebook to create 3D hailstone surface triangular mesh and output as an STL file. 
Includes code to add equidistant lobes to the hailstone, of custom length and width.

Method to calculate equidistant points on a sphere was from Deserno 2004: https://www.cmu.edu/biolphys/deserno/pdf/sphere_equi.pdf.
